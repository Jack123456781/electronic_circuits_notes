# -
电子电路课程的笔记

第七章 二极管  

这一章节主要讲的是  
1. 本征半导体，载流子等半导体基础概念
2. PN junction的形成以及它的内在原理,including 扩散运动，漂移运动
3. PN junction的单向导电性以及PN junction的伏安特性曲线，PN junction的电流方程
4. PN junction的击穿效应,including 雪崩击穿(低掺杂度，高电压即可击穿)，齐纳击穿(高掺杂度，低电压即可击穿)，以及不同击穿效应的杂糅对于稳压器的电压选择的作用
5. PN junction的电容效应,including 势垒电容，扩散电容......
6. 二极管的伏安特性曲线，温度对于二极管的影响及其伏安特性曲线
7. 二极管的主要参数(最大整流电流IF，最高反向通过电压UR，未击穿时的反向电流IR，最高工作频率fM)
8. 二极管的等效电路及二极管的应用：限幅，整流
